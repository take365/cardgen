あなたの物語を1枚のカードに。

ファンタジー／ビジネス／ミステリー……好きなテーマで誰でもカンタンにカード作成。
画像を選んで、テキストを入れて、保存するだけ。完全無料です。

今すぐカードを作る   GitHub（ソース公開）

改善要望・不具合は GitHub の Issues へ。
すべてに対応はできません。高度な加工は各自で AI や Canva 等をご利用ください。

メイン画面：プレビューしながらタイトル・本文・アイコンを編集 → PNG保存

はじめに（メインの画面）

  1. 背景・枠・アイテム・アイコンを選ぶ
  2. タイトル／本文を入力（プレビューを見ながら微調整）
  3. サイズを選んで PNG保存

※ 複数アイコン／追加テキストも追加可能。カードごとの雰囲気に合わせて調整してください。

作例（ファンタジー／ビジネス／ミステリー）

素材の選択（ローカル or Pixabay）

素材選択ダイアログ：ローカル画像 or Pixabay API 経由で検索

Pixabay の使い方（APIキー）

  1. Pixabay にログイン
  2. https://pixabay.com/api/docs/ を開く
  3. 画面上部の API Key をコピー
  4. 本ツールの「Pixabay」タブの入力欄に貼り付け → 検索して選択

Pixabay 素材は商用利用可能（要規約確認）。各画像のライセンス条件は Pixabay 側の規約に従います。

よくある質問

Q. 商用利用はできますか？
A. 生成された画像自体は本ツールとしては制限しませんが、使用する素材のライセンスに従ってください（Pixabay 等）。初期提供のサンプル素材はフリー素材としてご自由にご利用できます。

Q. 要望はどこに出せばいい？
A. GitHub Issues へどうぞ。全ての要望に対応できるわけではありません。複雑なデザインは各自で AI や Canva 等をご利用ください。
